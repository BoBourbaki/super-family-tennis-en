The current game mode is stored at CPU $00. Debug mode is game mode value $10. 
It can most easily be accessed by changing PRG offset $0131 from $0A to $10
Done with Game Genie code "FD67-0D0D"

Pointer table for debug option initialization code is stored from CPU $3EE7C-$3EE8B
	-There are 8 entries despite the debug mode only having 6 options. 
	 The last entry is repeated 2 times.
	-The 3rd entry is a repeat of the 2nd entry, so the 3rd debug option
	 "SCENARIO MESSAGE CHECK" is a copy of the one above it.

Cursor can be moved with d-pad and options selected with START
